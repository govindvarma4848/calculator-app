# Modern Digital Calculator

A modern responsive calculator built using HTML, CSS, and JavaScript.

## Features
- Responsive UI
- Glassmorphism design
- Keyboard support
- Arithmetic calculations
- Smooth animations
- Mobile friendly

## Tech Stack
- HTML5
- CSS3
- JavaScript
